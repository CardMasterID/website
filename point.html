Member Point CardMaster

Komunitas ini memiliki sistem point yang bertujuan sebagai tanda terimakasih karena telah berpartisipasi didalam komunitas kami, point diperoleh dari menjuarai challenge dan lainnya.

Monthly Challenge, Challenge ini diadakan pada setiap tanggal 1 dan 15, Event ini akan di urus oleh admin yang bertugas sebagai Event Regulator. Pemenang challenge akan mendapatkan ECM-Point yaitu 5 ECM-Point untuk juara 1, 4 ECM-Point untuk juara 2, Dan 3 ECM-Point untuk juara 3.

Daily Challenge, challenge ini diadakan di hari biasa Event ini akan di urus oleh admin yang bertugas sebagai Event Regulator. Pemenang challenge akan mendapatkan ECM-Point yaitu 3 ECM-Point untuk juara 1, 2 ECM-Point untuk juara 2, dan juga 1 ECM-Point untuk juara 3.
